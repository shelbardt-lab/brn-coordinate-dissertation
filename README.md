# Deciphering Southern Thailand's Violence

**Author:** Sascha Helbardt  
**Institution:** University of Passau  
**Year:** 2011

## Overview

This repository contains the doctoral dissertation:

*Deciphering Southern Thailand's Violence: Organisation and Insurgent Practices of BRN-Coordinate*

The dissertation analyzes:

- BRN-Coordinate
- Southern Thailand insurgency
- Recruitment and training
- Patani nationalism
- Political violence
- Counterinsurgency
- Communication and legitimacy

## Citation

Helbardt, Sascha. 2011. *Deciphering Southern Thailand's Violence: Organisation and Insurgent Practices of BRN-Coordinate*. Doctoral Dissertation, University of Passau.
